# Cybersecurity Toolkit

أداة شاملة ومتكاملة لاختبار الأمان السيبراني، توفر:

- **اكتشاف النطاقات الفرعية** باستخدام Sublist3r وAmass مع اختيار المستخدم.
- **فحص الثغرات الأمنية** مثل SQL Injection, XSS, Upload vulnerabilities, Admin panel bypass.
- **فحص شامل للخوادم والمواقع** باستخدام أدوات قوية مثل Nmap, httpx, dirsearch, nuclei, ffuf.
- **فحص سكربتات CMS** مثل وردبريس (WPScan) وجوملا (JoomScan).
- دعم تشغيل الأدوات **بشكل متوازي لتسريع الأداء**.
- **تثبيت تلقائي للأدوات** المطلوبة مع إدارة الاعتمادات.
---

## الأدوات المدمجة

| نوع الفحص               | الأدوات المستخدمة                                      |
|------------------------|-------------------------------------------------------|
| اكتشاف النطاقات الفرعية | Sublist3r, Amass                                      |
| فحص المنافذ والشبكات   | Nmap                                                  |
| فحص المسارات           | dirsearch, ffuf                                       |
| فحص الثغرات            | sqlmap (SQL Injection), XSStrike (XSS), admin-finder |
| فحص CMS                | WPScan (وردبريس), JoomScan (جوملا)                   |
| فحص القوالب الشاملة    | nuclei                                                |
| فحص إضافي              | httpx                                                 |

---

## متطلبات التشغيل

- Python 3.x
- أدوات النظام: git, snap, apt (على توزيعات Debian/Ubuntu)
- اتصال إنترنت لتثبيت الأدوات والاعتمادات

---

## كيفية الاستخدام

1. استنساخ المستودع أو تحميل السكربت.
2. تشغيل السكربت:


   ```bash
   python3 SubScanPro.py


3. إدخال النطاقات المستهدفة مفصولة بفواصل.
4. اختيار أداة اكتشاف النطاقات الفرعية.
5. انتظار النتائج التي تظهر مباشرة في واجهة الأداة.
6. مميزات إضافي
7. تشغيل الأدوات بالتوازي عبر ThreadPoolExecutor.
8. رسائل وواجهة ملونة لتسهيل متابعة الفحص.
9. تحديث تلقائي وتثبيت الأدوات الناقصة.
